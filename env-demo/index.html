Non-shared Secret Demo

This demo shows a Node.js app using a .env file (not in repo).

API_KEY=supersecret

require("dotenv").config();
console.log("API key:", process.env.API_KEY);

Best Practice: The .env file is listed in .gitignore and not pushed to the repo. You must create your own .env file to run the app.

See the GitHub repo for instructions.
